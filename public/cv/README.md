# CV / Resume

Coloca tu CV en PDF en esta carpeta con el nombre:

**Nicolas_Gonzalez_CV.pdf**

El botón "Descargar CV" en el navbar descargará automáticamente este archivo.

## Instrucciones:

1. Prepara tu CV en formato PDF
2. Nómbralo: `Nicolas_Gonzalez_CV.pdf`
3. Cópialo a esta carpeta (`public/cv/`)
4. El botón en el navbar ya está configurado para descargarlo

## Nota:
Si quieres cambiar el nombre del archivo, edita la línea 11 en:
`src/components/DownloadCV.jsx`
